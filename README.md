# Y Com (Rails)

Минималистичная соцсеть-прототип на Ruby on Rails 8 с авторизацией через Devise.

## Стек

- Ruby 3.4
- Rails 8.1
- PostgreSQL
- Devise

## Быстрый старт

1. Установить зависимости:

```bash
bundle install
```

2. Создать и накатить БД:

```bash
bin/rails db:create
bin/rails db:migrate
```

3. Запустить сервер:

```bash
bin/rails server
```

4. Открыть в браузере:

`http://localhost:3000`

## Основные маршруты

- `/auth` - вход/регистрация
- `/` - главная лента
- `/profile` - профиль пользователя
- `/settings` - настройки аккаунта и профиля
- `/messages`, `/following`, `/ai` - интерфейсные разделы

## Тесты

```bash
bin/rails test
```

## Что уже реализовано

- Аутентификация (Devise)
- Редактирование данных аккаунта из настроек:
  - имя
  - username
  - email
- Редактирование профиля:
  - имя
  - bio (до 160 символов)
- Базовые тесты контроллера и модели

## Замечания

- Часть экранов пока демонстрационные (много front-end интерактива на клиенте).
- При регистрации обязательны `name` и `username`.
